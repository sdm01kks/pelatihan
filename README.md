## 🚀 Cara Menjalankan Secara Lokal

1. **Clone repositori ini:**
   ```bash
   git clone https://github.com/sdm01kks/pelatihan.git
   cd pelatihan
   ```

2. **Buka langsung di browser:**
   ```bash
   # Cukup buka file index.html di browser favorit Anda
   open index.html
   ```
   atau klik dua kali pada file `index.html`.

---

## ✏️ Cara Memperbarui Konten

1. Edit file `index.html` sesuai kebutuhan
2. Commit dan push perubahan ke branch `main`
3. GitHub Pages akan otomatis memperbarui halaman dalam beberapa menit

---

## ⚙️ Mengaktifkan GitHub Pages

1. Masuk ke **Settings** repositori
2. Pilih menu **Pages** di sidebar kiri
3. Pada **Source**, pilih branch `main` dan folder `/ (root)`
4. Klik **Save** — halaman akan aktif di `https://sdm01kks.github.io/pelatihan`
